# Qirui Chen — Data Portfolio

Welcome! This repository collects my data analysis and machine learning projects.  
It’s organized to be clear for recruiters and collaborators: each project has a clean README, notebooks, and a results folder.

## Projects
- `projects/superstore-dashboard/` — EDA + Tableau dashboard
- `projects/credit-default/` — Classification model with explainability (SHAP)
- `projects/house-prices/` — Regression baseline vs. tree models

## Quick Start
```bash
# clone
git clone git@github.com:ciqrua/qirui-portfolio.git
cd qirui-portfolio

# create a Python env (optional but recommended)
python -m venv .venv && source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```
